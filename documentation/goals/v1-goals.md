## Version 1 Goals

The goal for v1 is to create a basic working meme editor that proves the core user flow works before adding more advanced features.

### V1 Core User Flow

A user should be able to:

1. Open the website.
2. Upload an image.
3. Click directly on the image to add text.
4. Move the text around the image.
5. Resize the text visually.
6. Change basic text styling.
7. Download or copy the finished meme.

### V1 Required Features

- Upload an image from the user's device.
- Click on the image to create editable text.
- Drag text to reposition it.
- Resize text without needing exact font-size numbers.
- Use Impact as the default meme font.
- Support basic fonts: Impact, Arial, Comic Sans, Helvetica, and Montserrat.
- Support white and black text colors.
- Support border, no border, and background highlight options.
- Allow the user to download the final meme.
- Make the website responsive enough for desktop, tablet, and phone screen sizes.

### V1 Stretch Goals

- Add a small meme template library with 5–10 templates.
- Add copy-to-clipboard support.
- Improve the visual design of the editor.

### Out of Scope for V1

The following features are not part of v1:

- Face swapping.
- Large meme library.
- Emotion-based meme search.
- Video or GIF editing.
- Direct sharing to social media.
- Audio clips or viral sounds.
- Advanced AI-based meme editing.
