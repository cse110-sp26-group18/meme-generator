# Prioritized Features/Tasks

[] Direct text-on-image editing
[] Compatibility across desktop, tablet, and phone
[] Meme templates with popular characters/emotions
[] Simple text tools:
  - Fonts
  - Colors
  - Borders/highlights
  - Resizeable text

# Feature Decisions

## High Priority
- Easy text editing directly on memes
- Mobile-friendly and responsive design
- Upload image/meme → click anywhere to add/move text
- Download or copy/paste generated memes
- Small meme template library (10–20 templates)

## Lower Priority
- Emotion-based meme search (for bigger library)
- Video/GIF text editing
- Direct social media sharing
- Audio clips/sounds on memes
- Face swap features

## Notes
- Start with photos before expanding to video
- Focus on speed and ease of use
- Meme templates can later expand into character/emotion categories
- AI text removal/editing on uploaded memes still needs testing
- Bigger library for future scalability like imgur/giphy
