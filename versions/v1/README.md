# Version v0.1

## Goal

Create the first working version of the meme editor.

## Planned Features

- Upload an image.
- Click on the image to add text.
- Move text around the image.
- Customize text style.
- Download the final meme.

## Notes

This version should focus on proving that the basic meme editing flow works before adding a meme library, AI features, video editing, or social sharing.
